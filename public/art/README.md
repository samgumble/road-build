# Groundwork title artwork

`groundwork-title-dawn.jpg` is the desktop title-screen hero. The mobile file is a center crop of
the same source, sized separately so phones do not download the desktop asset.

The source artwork was generated for this project with OpenAI's built-in image generation tool on
2026-07-10. It is original, contains no embedded text or third-party logos, and was prompted as a
premium low-poly island road-construction diorama at blue-hour dawn, using Groundwork's slate,
marine, moss, and safety-orange palette. Title text and controls remain HTML so they stay sharp,
accessible, responsive, and editable.
